1)  a+b*c ==> (a+(b*c))
          =>  (a+(bc*))
          =>  (a(bc*)+)
          =>  abc*+

2)  (p+q)*(r+s) => ((p+q)*(r+s))
                => ((pq+)*(r+s))      
                => ((pq+)*(rs+))
                => ((pq+)(rs+)*)
                => pq+rs+*   

3) a+b*c+d  ==> (((a+(b*c))+d))
            ==> (((a+(bc*))+d))
            ==> (((a(bc*)+)+d))
            ==> (((a(bc*)+)d+))      
            ==> abc*+d+
